Why learn ggplot2?

Base R can generate stunning plots . . . with time.

ggplot2 gets you to pretty plots very quickly. Also:

  • Consistent aesthetic.
  • Plays well with dplyr.
  • Systematic Syntax: "grammar of graphics".

Grammar of Graphics

Theory of data graphics proposed by Leland Wilkinson. Each statistical graphic has four elements:

  1. Data
  2. Mapping between variables and aesthetics (e.g. color, shape, scale)
  3. Geometric objects (e.g points, lines, polygons)
  4. Statistical transformation (e.g. smoothing, binning)

Two approaches to making a plot

  1. qplot
  2. ggplot + geom

We'll focus on ggplot + geom because it makes the grammar more transparent.

Before we get into plotting…

Let's subset our data so it is more manageable.

library(ggplot2)
library(dplyr)
set.seed(42)
small <- diamonds %>% sample_n(1000)

What is the relationship between the carat of diamonds and their price, when you're comparing diamonds of the same cut quality?

A look at the code

ggplot(small) 
+ geom_point(aes(x = carat, y = price, colour = cut)) 
+ scale_y_log10() 
+ facet_wrap(~cut) 
+ ggtitle("First example")

The various graphical components are separate blocks that are added together to make the plot.

Let's decompose that

  1. Data: ggplot(small)
  2. Layers: geom_point(aes(x = carat, y = price, colour = cut))
  3. facets: facet_wrap(~cut)
  4. scales: scale_y_log10()
  5. more options: ggtitle("First example")

Variations on syntax

ggplot(small) + geom_point(aes(x = carat, y = price, colour = cut))

versus

ggplot(small, aes(c = carat, y = price, colour = cut)) + geom_point()

A subtlety to be aware of

Caveat - there is a difference between setting and mapping aesthetics.

  1. Mapping dynamically links data to aesthetics through aes().
  2. Setting a parameter to a fixed value is done outside of aes.

Don't mix!

ggplot2 vs base

Strengths - Make beautiful graphics very fast. - Easy to update/modify plots. - Make highly customized plots very efficiently. - Plays great with dplyr.

Weaknesses - No 3d plotting or dynamic plots (see ggvis). - Can be difficult for unorthodox plots. - Slower processing time.
